In this project, I built a regression model to predict house prices using property features such as size, number of rooms, age, and furnished status. The workflow included loading and cleaning the dataset, handling missing values and categorical variables, training both Linear Regression and Random Forest models, and evaluating them with MAE, RMSE, and R². Visualizations like histograms, heatmaps, scatter plots, and boxplots helped uncover relationships between features and price.
The results showed that Linear Regression performed better than Random Forest, explaining about 65% of the variation in house prices. Key drivers of price were property size, number of rooms, and location-related features, while age had less influence than expected.

Libraries used:
NumPy → Numerical computations and array operations
Pandas → Data manipulation and preprocessing
Matplotlib → Data visualization (plots, charts)
Seaborn → Advanced statistical visualizations
Scikit-learn → Machine learning models, training, and evaluation
